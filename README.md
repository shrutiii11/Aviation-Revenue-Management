✈️ **Air India Aviation Sector Analysis – Power BI Project**
**Overview**
A Power BI dashboard project focused on analyzing airline operations, customer bookings, revenue trends, and route performance using aviation sector data inspired by Air India operations.

**Files**
aviation sector report.pbix – Main Power BI dashboard
Air India Project dataset.xlsx – Source dataset

**Key Features**
Flight delay and operational analysis
Revenue and booking insights
Passenger and loyalty analysis
Route performance tracking
Monthly and seasonal trend analysis

**Tools Used**
Power BI
Excel
DAX
Power Query

**Example KPIs**
Total Revenue
Average Delay Time
Passenger Load Factor
Booking Channel Performance

**Business Use Cases**
Improve operational efficiency
Analyze customer behavior
Optimize route profitability
Support business decision-making
